About

CorkPad

CorkPad is a Note Processing Desktop Enviroment that provides you with multiple tools to boost your productivity and time efficency.

CorkPad currently has three basic type of notes; Regular Notes, Checklist Notes and Custom Media Notes. In addition CorkPad also includes multiple widgets such as; Weather Forecast, Twitter feed, Spotify player, Timer Clock, Clicker Counter, etc.

CorkPad features a fully customizable environment. You can select between many of the included preset themes or customize every single section of the desktop to your likings. CorkPad currently allows you to change the note style (classic, modern, glassmorphism, neon, etc), the background (solid colors, gradients, textures, images, videos, custom links, etc.), navigation bar, etc.
